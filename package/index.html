Cloud Infrastructure

Compute

EC2 instances configured with custom AMIs, user-data bootstrapping, and auto-scaling across availability zones.

IAM roles and policies with least-privilege access. Bastion hosts for secure SSH tunneling into private subnets.

S3 buckets with versioning, lifecycle rules, and static website hosting. CloudFront for global distribution.

Nginx as a reverse proxy with TLS termination, rate limiting, and custom server blocks for multi-app deployments.
